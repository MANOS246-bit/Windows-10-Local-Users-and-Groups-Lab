# Networking Lab – Static IP, Ping & DNS

## 🛡️ Στόχος

Επαλήθευση διαφορών μεταξύ λογαριασμών Διαχειριστή (Administrator) και Τυπικού Χρήστη (Standard User) χρησιμοποιώντας το εργαλείο «Τοπικοί χρήστες και ομάδες» (Local Users & Groups):

* Δημιουργία λογαριασμού Διαχειριστή
* Δημιουργία λογαριασμού Τυπικού Χρήστη**
* Επαλήθευση της συμμετοχής σε ομάδες μέσω του εργαλείου «Τοπικοί χρήστες και ομάδες» (lusrmgr.msc).**
